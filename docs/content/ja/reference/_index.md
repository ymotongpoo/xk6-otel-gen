トポロジの記述方法とエクスポーターの設定に関するリファレンスです。

- [CLI ツール]({{< relref "/reference/cli-tools" >}}) — `xk6-otel-gen-viz`（インタラクティブ DAG 可視化）と `xk6-otel-gen-schema`（JSON Schema 出力）。
- [トポロジ YAML リファレンス]({{< relref "/reference/topology" >}}) — サービス、ジャーニー、障害、およびオペレーション単位のログイベント・メトリクス・プロファイル。
- [設定]({{< relref "/reference/configuration" >}}) — 優先順位、エンドポイント（プロファイル含む）、バッチ、エグゼンプラー、組み込みメトリクス。
